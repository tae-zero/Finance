# 🎮 주린이 놀이터 - 코스피 기업 분석 플랫폼

> **1차 프로젝트**: 초보 투자자를 위한 직관적이고 재미있는 주식 데이터 분석 플랫폼

## 🎯 프로젝트 개요

### 📋 프로젝트 요약
**주린이 놀이터**는 주식 초보자('주린이')들이 복잡한 금융 데이터를 놀이터처럼 재미있고 쉽게 탐색할 수 있도록 돕는 웹 기반 투자 분석 플랫폼입니다. 코스피 상장 기업들의 실시간 주식 데이터, 재무제표, 뉴스 정보를 게이미피케이션 요소와 함께 제공하여, 사용자가 직관적인 대시보드를 통해 기업의 투자 가치를 평가하고, 산업별 분석을 통해 시장 트렌드를 파악할 수 있습니다.

### 🎨 **최신 업데이트 (2024년 12월)**
- **완전한 UI/UX 리디자인**: Tech-Fin 스타일의 현대적이고 전문적인 디자인 시스템 적용
- **다크 모드 기본 적용**: 데이터 가독성을 극대화한 다크 테마
- **향상된 사용자 경험**: 8pt 그리드 시스템과 일관된 컴포넌트 디자인
- **게이미피케이션 강화**: 보물찾기 기능의 퀘스트형 인터페이스 개선

### 🎯 프로젝트 목표
- **데이터 기반 투자 의사결정 지원**: 실시간 주식 데이터와 재무제표 분석을 통한 객관적 투자 정보 제공
- **게이미피케이션을 통한 학습**: 복잡한 금융 지식을 재미있는 게임 요소로 변환하여 학습 효과 극대화
- **직관적 사용자 경험**: Tech-Fin 스타일의 현대적 디자인으로 전문성과 접근성의 균형 달성
- **종합적 기업 분석**: 단순한 주가 정보를 넘어서 재무상태, 뉴스, 산업 분석까지 통합 제공

## 🚀 주요 기능

### 📈 **대시보드 (Dashboard)**
- **실시간 코스피 지수 차트**: 1년간의 코스피 지수 변화를 시각화
- **핫뉴스 & 실적뉴스**: 최신 금융 뉴스와 실적 발표 뉴스 제공
- **투자자별 매매 현황**: 기관, 외국인, 개인 투자자별 매매 동향 분석
- **시가총액 TOP 10**: 시가총액 기준 상위 기업 리스트
- **거래량 TOP 5**: 거래량 기준 상위 종목 리스트

### 🏢 **기업 상세 분석 (Company Detail)**
- **기업 개요**: 기업명, 업종, 종목코드 등 기본 정보
- **주가 차트**: 3년간 주가 변화 추이
- **재무제표 분석**: 매출액, 영업이익률, DPS 등 핵심 재무지표
- **뉴스 분석**: 해당 기업 관련 최신 뉴스
- **투자자별 거래량**: 기관, 외국인, 개인 투자자별 매매 현황
- **종목 분석 리포트**: 전문가 분석 의견 및 목표주가

### 🏭 **산업별 분석 (Industry Analysis)**
- **산업 개요**: 각 산업의 특성과 전망
- **재무지표 해석**: 산업별 평균 재무지표와 해석
- **산업 체크포인트**: 투자 시 고려사항

### 🎮 **보물찾기 (Treasure Hunt)**
- **게이미피케이션 기반 종목 발굴**: 퀘스트와 미션 형태로 PER, PBR, ROE 등 기본 지표 활용
- **직관적 필터링**: 슬라이더를 통한 조건별 종목 검색 및 실시간 결과 업데이트
- **진행 상황 시각화**: 프로그레스 바, 뱃지, 보물상자 아이콘으로 성취감 제공
- **교육적 가이드**: 각 지표의 의미와 활용법을 게임 요소로 설명
